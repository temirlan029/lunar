Команды бота Lunar

Все команды работают только на целевом сервере.

1. /voice_stats
   - Показывает топ-10 участников по времени, проведенному в голосовых каналах.
   - Отправляет embed-отчет.

2. /voice_active
   - Показывает текущие активные голосовые сессии.
   - Отображает, кто сейчас находится в voice и сколько времени уже сидит.

3. /report
   - Отправляет текущий полный отчет по статистике голосовых каналов.
   - Удобно для ручной проверки в любой момент.

4. /set_report_channel
   - Устанавливает канал для автоматических отчетов каждые 12 часов.
   - Доступна только пользователям с правом управления сервером.

5. /reset
   - Сбрасывает статистику конкретного участника.
   - Нужно выбрать пользователя, которого надо обнулить.
   - Доступна только пользователям с правом управления сервером.

6. /stats_user
   - Показывает статистику конкретного участника.
   - Нужно выбрать пользователя, чтобы увидеть его время в голосе.

Примечание:
- Бот считает только время, проведенное в голосовых каналах на одном сервере.
- Переход между голосовыми каналами внутри сервера не сбрасывает сессию.
- При перезапуске бот сбрасывает текущие активные сессии и начинает отсчет заново с момента старта.
