Напиши код для Discord-бота на Python (используя библиотеку discord.py версии 2.4+), который отслеживает время, проведенное пользователями в голосовых каналах, и выводит отчет. Бот делается для одного конкретного сервера.

Технические требования к коду:

1. Подключение и интенты:
   - Использовать commands.Bot.
   - Включить discord.Intents.default(), discord.Intents.voice_states = True и discord.Intents.members = True.
   - Токен бота должен считываться из переменных окружения: os.environ.get("DISCORD_TOKEN").

2. База данных (SQLite):
   - Использовать встроенную библиотеку sqlite3.
   - При запуске бота автоматически создавать таблицу (если её нет), где будут храниться: user_id (INT, primary key) и total_time (REAL, общее время в секундах, по умолчанию 0).

3. Логика подсчета времени (on_voice_state_update):
   - Бот должен реагировать на вход пользователя в любой голосовой канал и запоминать точное время входа (например, в оперативной памяти в словаре active_sessions = {}).
   - При выходе пользователя из канала или переходе на другой сервер бот должен вычислять разницу во времени, переводить её в секунды и прибавлять к total_time этого пользователя в базе данных. После этого удалять сессию из словаря.
   - Переход из одного голосового канала в другой внутри одного сервера НЕ должен прерывать или сбрасывать счетчик времени (проверять, что before.channel != after.channel, но оба не None).
   - Игнорировать ботов.

4. Слэш-команды (App Commands / Hybrid Commands):
   - Создать одну слэш-команду /voice_stats, которая доступна всем пользователям.
   - Команда должна выводить красивое Embed-сообщение со списком топ-10 пользователей сервера по количеству набранного времени.
   - Время в отчете должно быть отформатировано в понятный вид: "Х ч. Y мин." (или "Y мин. Z сек.", если часов еще набралось меньше одного).

5. Синхронизация команд:
   - В событии on_ready добавить автоматическую синхронизацию слэш-команд (await bot.tree.sync()), чтобы они сразу появились на сервере.

Код должен быть чистым, с комментариями к ключевым блокам логики и готовым к деплою на хостинг.
